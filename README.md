# Project README

## Overview

This project aims to create an enhanced version of a linkhub or linktree, designed to be more dynamic and feature-rich. It allows for the creation of detailed profiles (Agents) with various personal and professional information, social media links, action buttons, and multimedia content.

## Features

- **Dynamic Agent Profiles**: Create profiles with comprehensive information such as name, role, description, profile pictures, and cover images.
- **Social Media Integration**: Include links to various social media platforms like WhatsApp, TikTok, Instagram, Facebook, LinkedIn, and YouTube.
- **Call to Action (CTA)**: Add call-to-action buttons that can link to files, external websites, or other resources.
- **Multimedia Support**: Incorporate videos and images to enhance the agent profiles.
- **Property Listings**: Display property details and link to more information.
- **Customizable Action Cards**: Create action cards with images, titles, and subtitles for additional engagement.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/your-repo-name.git
   ```

2. Navigate to the project directory:

   ```
   cd your-repo-name
   ```

3. Install dependencies:
   ```
   npm install
   # or
   yarn install
   ```

### Usage

1. Start the development server:

   ```
   npm start
   # or
   yarn start
   ```

2. Open your browser and go to `http://localhost:3000` to view the application.

### Creating a New Agent

To create a new agent, you need to add the agent's information to the `people` array in the `people.js` file.
